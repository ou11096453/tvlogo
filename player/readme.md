# Web-page-player (Enhanced Version)

基于 [xyz66882/Web-page-player](https://github.com/xyz66882/Web-page-player) 修改的增强版网页播放器。


---

## 🛠️ 安装与使用

1.  **部署环境**：
    *   推荐部署在 PHP 环境下（以使用 `proxy.php` 代理功能）。
    *   建议优先使用 **HTTP** 协议访问，以获得对所有协议视频源的最佳兼容性。
2.  **开始播放**：
    在地址栏输入视频 URL 或 M3U8 链接，点击播放即可。


---

## 📂 项目结构
*   `index.html`: 播放器主界面及 UI 逻辑。
*   `main.js`: 核心功能逻辑。
*   `proxy.php`: 处理跨域请求与代理转发的后端脚本。
*   `style.css`:  UI 样式表。
